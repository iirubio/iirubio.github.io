# Some ideas of how I'll do this

I'm currently doing these readings in my time off, which means I will probably struggle with one paper a day, so even though I call it that way, it will most likely be _at least 5 papers a week_. During my first weeks doing this, I'll be adjusting the format to see what fits better. Some ideas I currently have are:

* Adding keywords: some html templates I found already implement this
* Search bar
* A summary of each paper with the main takeaways
* 3 ideas of how the paper was writer, to improve my own writing
* 3 new research ideas that could come out from this paper. They might not actually new, but it at least forces me to think about ideas that can be expanded on from this paper (even if they are already expanded somewhere else)
* 3 papers cited that I should check out later
* Write down any methods used that I should check out
